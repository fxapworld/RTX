Manage packages

Build your catalog here. If your API on Render has GITHUB_TOKEN + ADMIN_ACCESS_KEY set, changes are committed to the repo automatically. Otherwise use Download products.json and commit manually. Host ZIPs on GitHub Releases / your CDN and paste the download URL.

Sign in: use the key from config.js → admin.accessKey (default is change-me — change it, commit, and use your new key). “Upload” files: upload your ZIP somewhere public (e.g. GitHub Release asset), then paste that link into Download URL for the package.
